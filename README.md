# Personal Portfolio Website

This repository contains the source code for my personal portfolio website, built to showcase my projects, skills, and contact information.

## Overview
The website is a responsive, front-end web application designed with a clean and simple layout to highlight my work as a computer science graduate. It serves as a central hub for my resume, GitHub, LinkedIn, and future blog content.

## Technologies Used
- React
- HTML
- CSS

## Features
- Responsive design optimized for desktop and mobile
- Modular React component structure
- Clear navigation and accessible layout
- Direct links to GitHub, LinkedIn, and email
- Scalable structure designed to support future blog and dynamic content features

## Motivation
I built this project to strengthen my front-end development skills and create a professional space to present my work to recruiters and collaborators!

- Add a blog section for technical writing
- Enhance interactivity and animations
- Improve accessibility and performance optimizations

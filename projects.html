Coffee Recommender System

This project is currently a Google Colab notebook, but eventually I would like to turn it into more, maybe a web app or something. It is intended to be a machine learning system that when given how a user likes their coffee, can recommend more coffee orders the user may also like. For example, if a user says they like their coffee with creaam, sugar, and caramel, the machine would recommend to give a caramel latte or butter pecan a try! This project is still in progress, and I hope to have it fully working soon.

Currently Working:

  • Collects taste preference data through a simple form
  • Uses Python with scikit-learn for recommendation algorithms

# Security Policy

## Scope

This is a personal portfolio website built with React and hosted on GitHub Pages. It does not handle authentication, store user data, or run a backend server. The only external service integration is a contact form powered by [Formspree](https://formspree.io).

## Reporting a Vulnerability

If you discover a security issue (e.g., XSS in rendered content, exposed credentials, or a dependency vulnerability), please report it by emailing **kbertozzi21@gmail.com** with the subject line "Security: kellybertozzi.github.io".

Please include:
- A description of the vulnerability
- Steps to reproduce the issue
- The potential impact

I will do my best to acknowledge your report within **48 hours** and address the issue promptly.

## Dependency Updates

This project uses Create React App and npm dependencies. I periodically review and update dependencies to patch known vulnerabilities. If you notice an outdated dependency with a known CVE, feel free to open an issue or submit a pull request.
